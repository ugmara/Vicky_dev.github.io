Why I Love JavaScript

JavaScript is the engine behind dynamic websites. It gives life to your frontend, makes pages interactive, and powers frameworks like React.

I started learning it with small click events and now I build full SPAs and fetch APIs. It’s the most empowering language I’ve used.

My Favorite CSS Tricks

CSS can feel like magic when you master it. One of my favorites is using clamp() for responsive font sizes. I also love using Flexbox and Grid for layouts.

Adding custom scroll behavior, theme transitions with CSS variables, and slick hover effects really makes a site stand out.
